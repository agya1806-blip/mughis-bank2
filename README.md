# MUGHIS BANK - ERP Mini

Aplikasi manajemen keuangan bisnis modern untuk Mughis Group.

## 🚀 Fitur

- ✅ Manajemen Keuangan (Pemasukan & Pengeluaran)
- ✅ Manajemen Invoice (3 jenis: Percetakan, Laptop, Umum)
- ✅ Manajemen Dompet (Multi-wallet support)
- ✅ Tracking Hutang & Piutang
- ✅ Laporan Keuangan (Harian, Mingguan, Bulanan, Tahunan)
- ✅ Dark Mode
- ✅ Offline Support (PWA)
- ✅ Cloud Sync (JSONBin)
- ✅ Export/Import Data

## 🌐 Demo

Akses live: https://mughis-bank.pages.dev

## 📦 Teknologi

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage API
- Service Worker (PWA)
- html2canvas (untuk capture invoice)

## 📱 Browser Support

- Chrome/Edge (Latest)
- Firefox (Latest)
- Safari (Latest)
- Mobile browsers
